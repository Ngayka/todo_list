# Todo List App

## Overview
This is a simple **Todo List application** built with **Django 6**. It allows users to manage tasks and tags in a clean, organized interface. The app supports creating, updating, deleting, and toggling the completion status of tasks, as well as managing tags.  

## Features

### Tasks
- List all tasks, ordered from not done → done, newest → oldest.
- Create new tasks.
- Edit or delete existing tasks.
- Mark tasks as **Complete** or **Undo** (toggle status).

### Tags
- List all tags.
- Create, update, and delete tags.
- Assign tags to tasks (optional feature if using task-tag relationships).

### User Interface
- Simple layout with a **sidebar** linking to Home (Task List) and Tag List.
- Responsive and easy to navigate.

## Tech Stack
- Python 3.12
- Django 6.0
- SQLite (default database)
- HTML/CSS (Django templates)
